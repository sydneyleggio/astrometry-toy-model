# astrometry-toy-model

C_parallelplot.py checks the shape of the parallel covariance matrix alone  

gamma_tilde_ab_sum.py sums Gamma tilde term and plots the Gamma curve as well as most often angular separation  

gammavstheta.py plots Gamma against different theta values  

hd_intermediate_sum.py computes the intermediate sum only for the purpose of writing it down similar to Romano - this is done in hd_full_curve.py now  

main.py plots the CP curve and its regime approximations  

hd_full_curve.py plots the HD curve and its regime approximations  

plot_cp_hd.py plots the CP and HD curves   

hd_full_matrix.py plots explicitly all 3 cases combined fully including weird jumps and errors
